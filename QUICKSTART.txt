============

1. Install Dependencies
   ---------------------
   npm install

   This will install:
   - electron
   - pdfkit
   - docx  
   - diff-match-patch (NEW!)


2. Run the Application
   -------------------
   npm start

TROUBLESHOOTING
===============

Issue: "Cannot find module 'diff-match-patch'"
Fix:   Run npm install

Issue: Streaming not working
Fix:   Check internet connection
       Verify API key is valid
       Try testing with shorter text first

Issue: Auto-save not recovering
Fix:   Check userData folder for draft_backup.json
       Ensure you waited 30+ seconds before crash

Issue: PDF formatting not applying
Fix:   Make sure you check the formatting options
       in the metadata modal before export
